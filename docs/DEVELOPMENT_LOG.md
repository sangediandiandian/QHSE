## 2026-07-15 · Stage 4A 通用审批流内核

计划提交：`build reusable approval workflow core`

完成内容：

- 新增通用审批实例和审批节点 Prisma 模型，以及内存/Prisma 双仓储。
- 支持按业务类型和业务 ID 关联流程、有序节点、节点角色白名单、审批通过、驳回和发起人撤回。

- 未暴露绕过领域权限的通用审批控制器，后续由各业务服务进行权限与审计封装。

验证记录：
